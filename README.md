# 🎓 Senior Year Digital Time Capsule

A Python-based persistent vault for high school memories. This project allows users to seal messages and senior quotes that remain "locked" until a specific date in the future.

## 🚀 Features
- **Time-Locking:** Messages cannot be read until the set `unlock_date`.
- **Obfuscation:** Quotes are stored in Base64 to prevent spoilers.
- **Persistence:** Memories are saved to a local JSON database.
- **OOP Design:** Built using Python classes for clean, modular code.

## 📅 Usage
- **Option 1:** Add a classmate's name, their superlative, and a secret message.
- **Option 2:** Check the vault. Locked messages will show a "Locked" status with the future date.
